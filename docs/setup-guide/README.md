# Setup Guide

Step-by-step guides for configuring Devin integrations in your environment.

## Guides

| Guide | Description | Status |
|---|---|---|
| API Key Setup | How to create a Devin service user and generate an API key | Planned |
| GitHub Actions Setup | Configuring repository secrets and workflow files | Planned |
| Azure DevOps Setup | Service hook configuration and webhook receiver deployment | Planned |
| Infrastructure Deployment | Deploying webhook infrastructure with Terraform or CloudFormation | Planned |
| Security Best Practices | Secret rotation, least-privilege access, network controls | Planned |

## Prerequisites

- A Devin account with API access
- Admin access to your CI/CD platform (GitHub, Azure DevOps, etc.)
- (For IaC deployments) CLI access to your cloud provider (AWS, Azure, GCP)
