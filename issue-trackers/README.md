# Issue Tracker Integrations for Devin

> Planned — not yet implemented.

Trigger Devin sessions from issue tracker transitions and events.

## Directories

| Directory | Description | Status |
|---|---|---|
| `linear/` | Linear webhook integration to trigger Devin on issue state changes | Planned |
| `servicenow/` | ServiceNow webhook integration to trigger Devin from change requests or incidents | Planned |

## Use Cases

- Trigger a Devin session when a Linear issue moves to "In Progress"
- Create Devin sessions from ServiceNow change requests for automated implementation
- Auto-generate investigation notes when ServiceNow incidents are filed
